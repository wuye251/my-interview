### 动态规划特点：

> __上次结果__决定/影响__本次结果__

### 动态规划解题步骤：

> 1. 确定dp数组（dp table）以及下标的含义
> 2. 确定递推公式
> 3. dp数组如何初始化
> 4. 确定遍历顺序
> 5. 举例推导dp数组

Step1: 分析题目，构建dp数组，确定数组行列对应的含义/情况

**Step2: 根据前面的结果，如果得出当前结果**

Step3: dp是否需要初始化？如何初始化？

Step4: 如何遍历，从前到后？从后到前？

Step5:  举例推导结果 是否和预期一致

---
